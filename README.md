# Structural-Textual KG Learning

This repository contains the experimental notebook for the research project:

**Integrating Structural and Textual Semantics for Ontology-Enriched Knowledge Graph Representation Learning**

The project explores a **KG–LM–KG framework**, technically implemented as a **KG-GNN–LM–GNN-KG pipeline** for knowledge graph representation learning and link prediction.

**The implemented pipeline follows this idea:**

Knowledge Graph → GNN Encoder → Frozen Language Model → GNN Refinement → Link Prediction
